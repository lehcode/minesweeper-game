# minesweeper-vue

1. Framework: Vue3
2. Styling: TailwindCSS
3. Bonus items:
   1. Auto-reveal functionality
   2. Reset button

## Requirements

Node.js >= v18.20.2

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
